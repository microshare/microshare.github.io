## S'inscrire pour un compte Microshare.io
---------------------------------------

### A. Développement ou production

Microshare utilise deux environnements : l'environnement de développement et l'environnement de production ; chacun a ses caractéristiques et répond à des besoins différents.

Dev : [dapp.microshare.io](https://dapp.microshare.io/login)

Prod : [app.microshare.io](https://app.microshare.io/login)

Voici les différences entre les deux :


| Dev                       | Prod                      |
| ------------------------- |:---------------------    :|
| Tableaux de bord de démo  | Compte client général     |
| API de démo               | Sauvegarde de données long terme |
| Tests                     |                           |
| Création de nouveaux tableaux de bord |              |


En bref, si Microshare vous invite à créer un compte, vous devrez créer un compte en production pour accéder à vos tableaux de bord. Si vous n'avez aucune obligation formelle et souhaitez simplement interagir avec l'environnement, le côté développement est mieux adapté à vos intentions.


### B. Inscription

<br>
**1.** Rendez-vous sur [https://app.microshare.io](https://app.microshare.io) ou [http://dapp.microshare.io](https://dapp.microshare.io/login).

<br>
{% include image.html url="/assets/img/create-microshare-account-1.png" description="Page de connexion" %}

**2.** Cliquez sur Sign Up.

<br>
{% include image.html url="/assets/img/create-microshare-account-2.png" description="Saisir une adresse e-mail" %}{% include image.html url="/assets/img/create-microshare-account-3.png" description="Fenêtre e-mail envoyé" %}

<br>
**3.** Vous recevrez un e-mail vous demandant de confirmer votre compte et de définir votre mot de passe.

<br>
{% include image.html url="/assets/img/create-microshare-account-4.png" description="E-mail de création de compte" %}{% include image.html url="/assets/img/create-microshare-account-5.png" description="Choisir un mot de passe" %}

<br>
**4.** Vous pourrez ensuite vous connecter à votre compte depuis [https://app.microshare.io](https://app.microshare.io) ou [http://dapp.microshare.io](https://dapp.microshare.io/login).

<br>
