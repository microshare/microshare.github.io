| Terme | Définition |
| ---------------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| | |
| 4th And Inches | Métaphore footballistique signifiant que vous êtes proche de la fin et devez tout donner pour atteindre l'objectif. |
| A Walk In The Park | Utilisé pour désigner une tâche simple. |
| A/B Testing | Test de deux versions (version A et version B) pour déterminer laquelle performe le mieux. |
| Accounts Payable | Registre de l'argent que vous devez aux personnes et entreprises qui ont contribué à créer votre produit ou service. |
| Accounts Receivable | Registre de l'argent que d'autres personnes et entreprises vous doivent. |
| Aggressive Timeline | Un calendrier trop court qui ne laisse pas assez de marge pour tout terminer. |
| Analytics | Données provenant de diverses sources utilisées pour orienter les efforts marketing. |
| B2B | Abréviation de business-to-business ; décrit une transaction commerciale entre entreprises. |
| B2C | Abréviation de business-to-consumer ; décrit les transactions avec des consommateurs individuels. |
| B2G | Abréviation de business-to-government ; décrit les transactions avec des entités gouvernementales. |
| Ballpark | Estimer quelque chose approximativement. |
| Be Proactive | Prendre l'initiative. |
| Benchmarking | Processus de mesure de divers aspects de vos systèmes (vitesse, efficacité, coût, volume de production, etc.). |
| BOFU | Bottom Of The Funnel — désigne les dernières étapes de l'entonnoir de vente, où le consommateur est prêt à acheter. |
| Bounce Rate | Fréquence à laquelle les visiteurs quittent votre site web sans cliquer sur quoi que ce soit. |
| Brand | Produit, identité ou image qui génère de la notoriété et distingue votre entreprise des autres. |
| Buyer Persona | Caractéristiques de votre acheteur idéal. |
| Capital | Le capital désigne souvent l'argent, mais peut aussi désigner tout ce que votre entreprise possède et utilise pour fonctionner (équipements, véhicules, bâtiments, terrains, etc.). |
| Cash Cow | Personne ou chose qui génère beaucoup d'argent. |
| Circle Back | On y reviendra si nous avons le temps. |
| Clean House | Licencier un grand groupe de personnes. |
| CMS | Abréviation de Content Management System ; programme (généralement logiciel) qui gère tous les aspects de la création de contenu numérique. |
| Conversion Rate | Pourcentage de personnes qui effectuent une action souhaitée (généralement sur votre site web). |
| Core Competency | Compétence ou avantage distinctif qui vous différencie de vos concurrents. |
| CPL | Abréviation de Cost Per Lead ; coût marketing total nécessaire pour acquérir un prospect. |
| CRM | Abréviation de Customer Relationship Management ; logiciel qui aide à organiser l'activité marketing. |
| CTR | Click Through Rate indique combien de personnes progressent sur votre site vers l'achat de votre produit ou service. |
| Deliverable | Produit ou service développé par une entreprise. |
| Demographics | Points de données applicables à votre marché cible, tels que l'âge, le sexe, le revenu et la situation familiale. |
| Digital Marketing | Marketing mené exclusivement sur Internet. |
| Don't Go There | Nous n'en discutons pas pour le moment. |
| Evergreen | Contenu utile pour un consommateur quelle que soit la date à laquelle il est consulté. |
| Fixed Costs | Coûts à payer que l'entreprise aille bien ou mal (loyer, salaires, services publics, etc.). |
| Friction | Tout aspect de votre image, marque, produit ou site web difficile à comprendre (crée une friction entre celui-ci et le consommateur). |
| Go Back And Sharpen Your Pencils | Trouver de nouvelles idées. |
| Gross | Brut ; montant ou quantité totale AVANT déductions ou dépenses. |
| Housekeeping | Questions organisationnelles et de projet routinières. |
| Inbound Marketing | Marketing numérique incluant podcasts, vidéo, e-mails, réseaux sociaux, livres électroniques et SEO. |
| Incentivize | Fournir une incitation (motivation) à l'utilisation d'un produit ou service. |
| Infographic | Contenu combinant texte et images pour rendre des informations complexes faciles à comprendre. |
| KPI | Acronyme de Key Performance Indicators ; indicateurs chiffrés mesurant l'efficacité de l'entreprise dans un domaine donné. |
| Live And Breathe It 24/7/365 | Travailler sur quelque chose en permanence au détriment de tout le reste. |
| Loop You In | Inclure dans la discussion. |
| Manage The Optics | Présenter les faits de manière favorable. |
| Margin | Profit d'un produit ou service après déduction de toutes les dépenses ; souvent exprimé en pourcentage. |
| Market Penetration | Mesure de l'étendue des ventes d'un produit par rapport au volume total des ventes de tous les produits concurrents. |
| Market Research | Action ou activité de collecte d'informations sur les besoins et préférences des consommateurs pour proposer le bon produit ou service. |
| Marketing | Action ou activité de promotion et de vente de produits ou services. |
| Metrics | Toute mesure quantifiable utilisée par l'entreprise pour évaluer la performance. |
| MOFU | Middle Of The Funnel — étapes intermédiaires de l'entonnoir de vente, où l'entreprise se positionne comme solution au problème du consommateur. |
| Monetize | Générer de l'argent à partir d'un produit ou d'une activité. |
| Net | Net ; montant ou quantité APRÈS déductions ou dépenses. |
| Niche Market | Segment très spécifique d'un marché plus large. |
| Not Enough Boots On The Ground | Pas assez de personnes travaillant sur une tâche. |
| PPC | Pay Per Click — publicité sur Internet où vous ne payez que lorsqu'une personne clique sur votre annonce. |
| R&D | Abréviation de research and development (recherche et développement). |
| Responsive Design | Site web qui s'adapte au type d'appareil (tablette, téléphone, ordinateur portable, bureau) utilisé pour le consulter. |
| ROI | Return on investment ; ensemble des bénéfices — monétaires ou autres — obtenus grâce à un investissement. |
| Run With It | Prendre une idée et l'approfondir. |
| Sales Funnel | L'ensemble du processus de vente. |
| Scalable | Capable d'être modifié en taille. |
| SEO | Search Engine Optimization — optimisation du site web pour un meilleur classement dans les résultats des moteurs de recherche. |
| Square The Circle | Faire les choses différemment. |
| SWOT | Acronyme d'une analyse examinant : * Forces * Faiblesses * Opportunités * Menaces |
| Take It To The Next Level | Intensifier les efforts. |
| Think Outside The Box | Faire preuve de créativité. |
| TOFU | Top Of The Funnel — premières étapes de l'entonnoir de vente, où le consommateur cherche des réponses à un problème pouvant impliquer votre produit ou service. |
| Unique Selling Proposition | Facteur spécifique qui différencie votre produit ou service de celui d'un concurrent direct (coût, qualité, usage supplémentaire, etc.). |
| User Experience | Expérience totale — de l'achat et au-delà — qu'un utilisateur a avec votre marque. |
| Variable Costs | Dépenses qui fluctuent selon le volume d'activité : * Expédition * Commissions * Fournitures * Salaires horaires |
