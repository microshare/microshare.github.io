### Comment installer un capteur TABS Motion    
     
1.  Activez le capteur en retirant la languette isolante de la batterie du capteur.
   {% include image.html url="/assets/img/install-tabs-motion/image1.png" width=400 description="Banner" %}

2.  Appliquez le ruban adhésif sur le côté du capteur qui ne comporte PAS le QR code.
   {% include image.html url="/assets/img/install-tabs-motion/image2.png" width=400 description="Banner" %}

3. Placez le capuchon sur l'ampoule du capteur avec le côté ouvert orienté vers le côté adhésif de l'appareil.
   {% include image.html url="/assets/img/install-tabs-motion/image3.png" width=400 description="Banner" %}

4.  Connectez-vous à l'application Deploy-M2 avec vos identifiants de compte.
   {% include image.html url="/assets/img/install-tabs-motion/image4.png" width=400 description="Banner" %}

5.  Sélectionnez le Device Cluster approprié pour l'appareil que vous installez.
  **Remarque :** un Device Cluster est simplement un regroupement de capteurs par type de capteur et lieu d'installation.
   {% include image.html url="/assets/img/install-tabs-motion/image5.png" width=400 description="Banner" %}

6.  Si des tags ont été créés à l'avance, sélectionnez l'emplacement où l'appareil sera installé.

7.  Cliquez sur l'icône Appareil photo.
   {% include image.html url="/assets/img/install-tabs-motion/image6.png" width=400 description="Banner" %}

8.  Scannez le QR code sur le capteur.
   {% include image.html url="/assets/img/install-tabs-motion/image7.png" width=400 description="Banner" %}

9.  Saisissez les tags d'emplacement appropriés.
   {% include image.html url="/assets/img/install-tabs-motion/image8.png" width=400 description="Banner" %}

10. Confirmez que le DevEUI du capteur est maintenant associé à l'emplacement dans Deploy-M.

11. Sélectionnez Enregistrer en bas de l'écran.
   {% include image.html url="/assets/img/install-tabs-motion/image9.png" width=400 description="Banner" %}

12. Retirez le film protecteur de l'adhésif et fixez le capteur en haut de l'encadrement de porte, l'élément de détection orienté vers le bas et le QR code tourné vers vous.
   {% include image.html url="/assets/img/install-tabs-motion/image10.png" width=400 description="Banner" %}

13. Maintenez le capteur contre l'encadrement pendant 30 secondes pour permettre à l'adhésif de bien adhérer.
