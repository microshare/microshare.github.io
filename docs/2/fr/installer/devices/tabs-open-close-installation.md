### Comment installer un capteur TABS Open/Close 

1.  Activez le capteur en retirant la languette isolante de la batterie du capteur.
   {% include image.html url="/assets/img/install-tabs-open-close/image1.png" width=400 description="Banner" %}

2.  Appliquez le ruban adhésif sur le côté du capteur qui ne comporte PAS le QR code.
   {% include image.html url="/assets/img/install-tabs-open-close/image2.png" width=400 description="Banner" %}

3.  Connectez-vous à l'application Deploy-M2 avec vos identifiants de compte.
   {% include image.html url="/assets/img/install-tabs-open-close/image3.png" width=400 description="Banner" %}

4.  Sélectionnez le Device Cluster approprié pour l'appareil que vous installez.
    **Remarque :** un Device Cluster est simplement un regroupement de capteurs par type de capteur et lieu d'installation.
   {% include image.html url="/assets/img/install-tabs-open-close/image4.png" width=400 description="Banner" %}

5.  Si des tags ont été créés à l'avance, sélectionnez l'emplacement où l'appareil sera installé.

6.  Cliquez sur l'icône Appareil photo.
   {% include image.html url="/assets/img/install-tabs-open-close/image5.png" width=400 description="Banner" %}

7.  Scannez le QR code sur le capteur.
   {% include image.html url="/assets/img/install-tabs-open-close/image6.png" width=400 description="Banner" %}

8.  Saisissez les tags d'emplacement appropriés.
   {% include image.html url="/assets/img/install-tabs-open-close/image7.png" width=400 description="Banner" %}

9.  Confirmez que le DevEUI du capteur est maintenant associé à l'emplacement dans Deploy-M.

10. Sélectionnez Enregistrer en bas de l'écran.
   {% include image.html url="/assets/img/install-tabs-open-close/image8.png" width=400 description="Banner" %}

11. Retirez le film protecteur de l'adhésif et appuyez l'appareil contre le mur à côté de l'ouverture que vous souhaitez surveiller.

12. Fixez l'aimant sur la porte ou la fenêtre à proximité du capteur, en veillant à ce qu'ils soient alignés.
   {% include image.html url="/assets/img/install-tabs-open-close/image9.png" width=400 description="Banner" %}

13. Maintenez le capteur contre l'encadrement pendant 30 secondes pour permettre à l'adhésif de bien adhérer.
