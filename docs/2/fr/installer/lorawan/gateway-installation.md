## 1. Introduction
---------------------------------------

Les passerelles font le lien entre vos appareils et Internet. Les appareils utilisent des réseaux LoRaWAN basse consommation pour se connecter à la passerelle, tandis que la passerelle utilise des réseaux à haut débit comme le WiFi, l'Ethernet ou le cellulaire pour se connecter au cloud.
