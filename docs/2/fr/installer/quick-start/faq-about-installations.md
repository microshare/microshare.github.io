{% capture title %}
Où se trouve le capteur de mouvement Tabs sur mon appareil ?
{% endcapture %}

{% capture content %}
Le capteur de mouvement Tabs possède un capteur infrarouge sur le côté de l'appareil. Ce capteur est utilisé dans les solutions d'occupation de salle, d'occupation de bureau et de détection de mouvement.
<br><br>

{% capture title %}
Comment faire la différence entre le capteur Tabs Température/Humidité et le capteur de qualité de l'air ?
{% endcapture %}

{% capture content %}
Pour différencier vos appareils, regardez les informations au dos. Le capteur de qualité de l'air porte le numéro de modèle « TBHV110 » et le capteur température/humidité porte le numéro de modèle « TBHH100 ».
<br>
