Pour cette partie, nous supposerons que vous êtes à l'aise avec l'interaction avec les API Microshare et la plateforme Microshare..

Voici une collection de requêtes plus complète : [![Run in Postman](https://run.pstmn.io/button.svg)](https://app.getpostman.com/run-collection/800c4698e3342072364c)
