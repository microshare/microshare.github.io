Le logiciel Arduino open source (IDE) facilite l'écriture de code et son téléversement sur la carte. Il fonctionne sous Windows, Mac OS X et Linux. L'environnement est écrit en Java et s'appuie sur Processing et d'autres logiciels open source.

## Télécharger [Arduino Desktop IDE](https://www.arduino.cc/en/Guide/HomePage){:target="_blank"}

Téléchargez l'Arduino Desktop IDE pour votre plateforme Linux, Mac OSX ou Windows. N'utilisez pas la version Web Editor pour cet exemple. Des conflits de bibliothèques surviennent avec la version Web Editor.

## Étapes suivantes

Maintenant que vous avez installé l'Arduino Desktop IDE, vous pouvez consulter l'un de nos tutoriels de programmation d'appareils et apprendre à programmer vos dispositifs.
