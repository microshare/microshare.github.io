## Analyser, étiqueter et installer un capteur

Ce guide vous explique comment ajouter un nouveau capteur à votre groupe d'appareils dans DeployM, en scannant son code QR, en configurant ses balises et en l'installant physiquement.

---

## Étapes

### 1. Sélectionner le groupe d'appareils

Appuyez sur le groupe d'appareils qui contiendra tous vos capteurs.

{% include image.html url="/assets/img/uploads/demo/pdf-page-1-img-1.png" description="Écran d'accueil – liste des groupes d'appareils (Device Clusters)" %}

---

### 2. Ajouter un nouvel appareil via le bouton Scan

Lorsque vous souhaitez créer une nouvelle entrée plutôt que de modifier une entrée existante, appuyez sur le bouton **Scan** depuis la page du groupe d'appareils pour ajouter un nouvel appareil.

{% include image.html url="/assets/img/uploads/demo/pdf-page-2-img-1.png" description="Page du groupe d'appareils avec le bouton Scan et la liste des appareils existants" %}

---

### 3. Ouverture de la caméra en mode scan

Comme pour la passerelle, le bouton **Scan** ouvre votre caméra en mode scan.

{% include image.html url="/assets/img/uploads/demo/pdf-page-2-img-2.png" description="Caméra ouverte en mode scan QR Code" %}

---

### 4. Scanner le code QR du capteur

Un autocollant avec un code QR est apposé sur votre capteur. Placez-le dans le cadre et rapprochez-vous suffisamment pour que votre caméra puisse bien le voir. Vous aurez peut-être besoin de plus de lumière si la caméra ne parvient pas à détecter le code.

{% include image.html url="/assets/img/uploads/demo/pdf-page-3-img-1.png" description="Scan du code QR apposé sur le capteur" %}

---

### 5. Saisir les balises du capteur

Pour un nouveau capteur, aucune balise n'est définie. Ajoutons des balises pour indiquer où ce piège sera déployé. Commencez par appuyer sur le champ de balise vide.

{% include image.html url="/assets/img/uploads/demo/pdf-page-3-img-2.png" description="Formulaire d'étiquetage du capteur – champs de balises vides" %}

---

### 6. Sélectionner l'option de balise personnalisée

Sélectionnez l'option permettant de saisir une balise personnalisée.

> **Remarque :** Pour les futurs appareils, les balises précédemment utilisées s'afficheront automatiquement afin de vous permettre de les sélectionner rapidement et de garder vos déploiements bien organisés.

{% include image.html url="/assets/img/uploads/demo/pdf-page-4-img-1.png" description="Sélection de l'option de saisie d'une balise personnalisée" %}

---

### 7. Saisir le tag personnalisé

Saisissez le tag personnalisé que vous souhaitez utiliser, puis appuyez sur **Terminé**.

{% include image.html url="/assets/img/uploads/demo/pdf-page-4-img-2.png" description="Saisie du tag personnalisé via le clavier" %}

---

### 8. Répéter pour chaque balise

Répétez cette procédure pour chacune des balises jusqu'à ce que vous ayez défini une valeur pour chacune d'entre elles.

{% include image.html url="/assets/img/uploads/demo/pdf-page-5-img-1.png" description="Formulaire d'étiquetage complété – toutes les balises renseignées" %}

---

### 9. Enregistrer les modifications

Une fois que vous avez terminé, cliquez sur le bouton **Enregistrer** pour sauvegarder vos modifications.

{% include image.html url="/assets/img/uploads/demo/pdf-page-5-img-2.png" description="Bouton Enregistrer mis en évidence pour sauvegarder les balises" %}

---

### 10. Vérifier l'ajout du capteur

Vous pouvez voir que votre premier capteur a été ajouté avec succès. Appuyez sur le bouton **Retour** pour revenir à votre liste.

{% include image.html url="/assets/img/uploads/demo/pdf-page-6-img-1.png" description="Confirmation de l'ajout du capteur dans la liste du groupe d'appareils" %}

---

### 11. Installer le capteur physiquement

Une fois étiqueté, vous pouvez installer le capteur en retirant l'autocollant de la face adhésive et en fixant l'appareil à l'emplacement approprié ou sur le panier d'appât. Veillez à ce que la lentille du capteur soit orientée vers la zone que vous souhaitez surveiller.

---

### 12. Retour à l'écran principal DeployM

C'est tout pour le moment. Vous pouvez retourner sur DeployM pour ajouter ou modifier vos passerelles et vos capteurs en scannant les codes QR et en configurant les balises.

{% include image.html url="/assets/img/uploads/demo/pdf-page-7-img-1.png" description="Écran principal DeployM – vue d'ensemble des groupes d'appareils configurés" %}
