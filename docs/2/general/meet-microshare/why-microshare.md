## Analyser, étiqueter et installer un capteur

Ce guide vous explique comment ajouter un nouveau capteur à un groupe d'appareils existant, lui attribuer des balises de localisation et l'installer physiquement sur site.

---

## Étapes

### Étape 1 — Sélectionner le groupe d'appareils

Appuyez sur le groupe d'appareils qui contiendra tous vos capteurs.

{% include image.html url="/assets/img/uploads/why-microshare/pdf-page-1-img-1.png" description="Écran Device Clusters — sélectionner le groupe d'appareils cible" %}

---

### Étape 2 — Ajouter un nouvel appareil via le bouton Scan

Lorsque vous souhaitez créer une nouvelle entrée plutôt que de modifier une entrée existante, appuyez sur le bouton **Scan** depuis la page du groupe d'appareils pour ajouter un nouvel appareil.

{% include image.html url="/assets/img/uploads/why-microshare/pdf-page-2-img-1.png" description="Page du groupe d'appareils avec le bouton Scan pour ajouter un nouvel appareil" %}

---

### Étape 3 — Ouverture de la caméra en mode scan

Comme pour la passerelle, le bouton **Scan** ouvre votre caméra en mode scan.

{% include image.html url="/assets/img/uploads/why-microshare/pdf-page-2-img-2.png" description="Caméra ouverte en mode scan — prête à lire un code QR" %}

---

### Étape 4 — Scanner le code QR du capteur

Un autocollant avec un code QR est apposé sur votre capteur. Placez-le dans le cadre et rapprochez-vous suffisamment pour que votre caméra puisse bien le voir. Vous aurez peut-être besoin de plus de lumière si la caméra ne parvient pas à détecter le code.

{% include image.html url="/assets/img/uploads/why-microshare/pdf-page-3-img-1.png" description="Scan du code QR apposé sur le capteur" %}

---

### Étape 5 — Ajouter des balises au capteur

Pour un nouveau capteur, aucune balise n'est définie. Ajoutez des balises pour indiquer où ce capteur (ou piège) sera déployé. Commencez par appuyer sur le champ de balise vide.

{% include image.html url="/assets/img/uploads/why-microshare/pdf-page-3-img-2.png" description="Formulaire de configuration du capteur — champs de balises vides" %}

---

### Étape 6 — Saisir une balise personnalisée

Sélectionnez l'option permettant de saisir une balise personnalisée.

> **Remarque :** Pour les futurs appareils, les balises précédemment utilisées s'afficheront automatiquement afin que vous puissiez les sélectionner rapidement et maintenir vos déploiements bien organisés.

{% include image.html url="/assets/img/uploads/why-microshare/pdf-page-4-img-1.png" description="Sélection de l'option de balise personnalisée dans le menu Edit Site" %}

---

### Étape 7 — Confirmer la balise saisie

Saisissez le tag personnalisé que vous souhaitez utiliser, puis appuyez sur **Terminé**.

{% include image.html url="/assets/img/uploads/why-microshare/pdf-page-4-img-2.png" description="Saisie du tag personnalisé via le clavier et confirmation" %}

---

### Étape 8 — Répéter pour chaque balise

Répétez cette procédure pour chacune des balises jusqu'à ce que vous ayez défini une valeur pour chacune d'entre elles.

Les champs de balises disponibles sont les suivants :

- **Customer**
- **Site**
- **Area**
- **Sensor #**
- **Placement**
- **Configuration**

{% include image.html url="/assets/img/uploads/why-microshare/pdf-page-5-img-1.png" description="Tous les champs de balises renseignés — Customer, Site, Area, Sensor #, Placement, Configuration" %}

---

### Étape 9 — Enregistrer les modifications

Une fois que vous avez terminé de renseigner toutes les balises, cliquez sur le bouton **Enregistrer** (Save) pour sauvegarder vos modifications.

{% include image.html url="/assets/img/uploads/why-microshare/pdf-page-5-img-2.png" description="Clic sur le bouton Save pour enregistrer la configuration du capteur" %}

---

### Étape 10 — Vérifier l'ajout du capteur

Vous pouvez voir que votre premier capteur a été ajouté avec succès. Appuyez sur le bouton **Retour** (Back) pour revenir à votre liste de capteurs.

{% include image.html url="/assets/img/uploads/why-microshare/pdf-page-6-img-1.png" description="Confirmation de l'ajout du capteur dans la liste des appareils" %}

---

### Étape 11 — Installer physiquement le capteur

Une fois étiqueté, vous pouvez installer le capteur en retirant l'autocollant de la face adhésive et en fixant l'appareil à l'emplacement approprié ou sur le panier d'appât.

> **Important :** Veillez à ce que la lentille du capteur soit orientée vers la zone que vous souhaitez surveiller.

---

### Étape 12 — Retour à l'écran principal DeployM

C'est tout pour le moment. Vous pouvez retourner sur **DeployM** pour ajouter ou modifier vos passerelles et vos capteurs en scannant les codes QR et en configurant les balises.

{% include image.html url="/assets/img/uploads/why-microshare/pdf-page-7-img-1.png" description="Écran principal DeployM affichant les groupes d'appareils configurés" %}

---

## Résumé

| Étape | Action |
|---|---|
| 1 | Sélectionner le groupe d'appareils |
| 2–3 | Appuyer sur **Scan** pour ouvrir la caméra |
| 4 | Scanner le code QR du capteur |
| 5–8 | Renseigner les balises (Customer, Site, Area, etc.) |
| 9 | Enregistrer la configuration |
| 10 | Vérifier l'ajout dans la liste |
| 11 | Installer physiquement le capteur |
| 12 | Retourner sur DeployM pour tout autre modification |
